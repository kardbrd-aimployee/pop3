  <action>
    Create src/engine/objects/cell_grid.rs with:

    1. `pub const CELL_GRID_SIZE: usize = 128;`
       `pub const CELL_GRID_TOTAL: usize = CELL_GRID_SIZE * CELL_GRID_SIZE;` (16384 cells)

    2. `pub struct CellGrid`:
       - `heads: Box<[Option<u16>; CELL_GRID_TOTAL]>` — first_object handle per cell
       - This is a SEPARATE structure from RegionMap (per user decision: "minimize disruption" to RegionMap, and per research recommendation about keeping cell grid separate to avoid repr(C) layout issues)

    3. `CellGrid::new()` — all heads = None

    4. `CellGrid::insert_object(&mut self, handle: u16, cell_idx: usize, slots: &mut [PoolSlot])`:
       - Set new object's next_in_cell = current head, prev_in_cell = None
       - If old head exists, set old head's prev_in_cell = handle
       - Set cell head = Some(handle)

    5. `CellGrid::remove_object(&mut self, handle: u16, cell_idx: usize, slots: &mut [PoolSlot])`:
       - Read object's prev/next
       - If has prev: prev.next = next; else: cell head = next
       - If has next: next.prev = prev
       - Clear object's prev_in_cell and next_in_cell to None

    6. `CellGrid::set_position(handle, old_pos: WorldCoord, new_pos: WorldCoord, slots: &mut [PoolSlot])`:
       - Compute old_cell = old_pos.to_tile().cell_index()
       - Compute new_cell = new_pos.to_tile().cell_index()
       - If old_cell == new_cell: return (no-op)
       - remove_object(handle, old_cell, slots)
       - insert_object(handle, new_cell, slots)

    7. `CellGrid::cell_head(&self, cell_idx: usize) -> Option<u16>` — read the head for a cell

    8. Helper: `CellGrid::cell_index_from_world(pos: &WorldCoord) -> usize` — convenience wrapper around to_tile().cell_index()

    Update mod.rs to add `pub mod cell_grid;` and `pub use cell_grid::{CellGrid, CELL_GRID_SIZE};`

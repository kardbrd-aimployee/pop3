```rust
use crate::engine::movement::constants::REGION_GRID_SIZE;
use super::types::PoolSlot;
use super::handle::ObjectHandle;
```

**CellObjectHeads struct:**
- `heads: Vec<Option<u16>>` with length `REGION_GRID_SIZE * REGION_GRID_SIZE` (128*128 = 16384 cells)
- All initialized to `None`

**Methods:**

`new() -> Self` -- create with all heads = None.

`insert_object(&mut self, pool_slots: &mut [PoolSlot], handle: ObjectHandle, cell_idx: usize)`:
- Get old head from `self.heads[cell_idx]`
- Access `pool_slots[handle as usize]` as Occupied, set `header.next_in_cell = old_head`, `header.prev_in_cell = None`
- If old_head is Some, access that slot's header and set `prev_in_cell = Some(handle)`
- Set `self.heads[cell_idx] = Some(handle)`

`remove_object(&mut self, pool_slots: &mut [PoolSlot], handle: ObjectHandle, cell_idx: usize)`:
- Read prev and next from the object's header (next_in_cell, prev_in_cell)
- If prev is Some: set prev's `next_in_cell = next`. Else: set `self.heads[cell_idx] = next` (was head)
- If next is Some: set next's `prev_in_cell = prev`
- Clear the removed object's `next_in_cell` and `prev_in_cell` to None

`cell_objects<'a>(&'a self, pool_slots: &'a [PoolSlot], cell_idx: usize) -> CellObjectIter<'a>`:
- Return iterator starting from `self.heads[cell_idx]`, following `next_in_cell` chain

**CellObjectIter struct:**
- Holds `&[PoolSlot]` and `current: Option<u16>`
- Iterator yields `ObjectHandle` by following next_in_cell chain

Add `pub mod cell_grid;` and `pub use cell_grid::CellObjectHeads;` to `src/engine/objects/mod.rs`.

Write all tests inline in `#[cfg(test)] mod tests {}` using pool.create() to allocate objects at known positions, then calling insert/remove directly.

Create `src/engine/objects/pool.rs` with:

Constants:
```rust
pub const MAX_OBJECTS: usize = 1101;       // 0x44D
pub const LOW_PRIORITY_START: usize = 461; // High: 0..461, Low: 461..1101
```

`ObjectPool` struct with fields:
- `slots: Box<[PoolSlot; MAX_OBJECTS]>` -- heap-allocated to avoid stack overflow
- `free_high: Option<u16>` -- head of high-priority free list (slots 0..461)
- `free_low: Option<u16>` -- head of low-priority free list (slots 461..1101)
- `active_count: u16`

Methods:

`new()` -- Initialize all slots as `PoolSlot::Free`. Link high-priority: slot 0 next=1, slot 1 next=2, ..., slot 460 next=None. `free_high = Some(0)`. Link low-priority: slot 461 next=462, ..., slot 1100 next=None. `free_low = Some(461)`. `active_count = 0`.

Use Vec to build then convert to Box array (same pattern from research):
```rust
let mut v = Vec::with_capacity(MAX_OBJECTS);
for _ in 0..MAX_OBJECTS { v.push(PoolSlot::Free { next_free: None }); }
let mut slots: Box<[PoolSlot; MAX_OBJECTS]> = v.into_boxed_slice().try_into().unwrap();
```

`create(model_type: ModelType, subtype: u8, tribe: u8, position: WorldCoord) -> Option<ObjectHandle>`:
- Determine priority: Person, Building, Creature, Vehicle, Scenery = high; General, Effect, Shot, Shape, Internal, Spell = low
- Pop from appropriate free list head
- Construct ObjectHeader with all fields zeroed except model_type, subtype, tribe, position, object_index=slot_idx
- Construct GameObjectData based on model_type (Person -> PersonData::default(), others -> empty tuple stubs)
- Set slot to Occupied, increment active_count
- Return Some(slot_idx)

`destroy(handle: ObjectHandle) -> bool`:
- Bounds check handle < MAX_OBJECTS
- Verify slot is Occupied (return false if Free)
- Determine priority from slot index (< LOW_PRIORITY_START = high)
- Push slot to front of appropriate free list (set slot to Free { next_free: current_head }, update head)
- Decrement active_count
- Return true

`get(handle: ObjectHandle) -> Option<&GameObject>`:
- Bounds check, return reference if Occupied, None if Free

`get_mut(handle: ObjectHandle) -> Option<&mut GameObject>`:
- Same as get but mutable

`persons(&self) -> impl Iterator<Item = (ObjectHandle, &ObjectHeader, &PersonData)>`:
- Iterate all slots, filter for Occupied with Person variant, yield (index, header, person_data)

`persons_mut(&mut self) -> impl Iterator<Item = (ObjectHandle, &mut ObjectHeader, &mut PersonData)>`:
- Same but mutable

`active_count(&self) -> u16`:
- Return self.active_count

Write all tests inline in `#[cfg(test)] mod tests {}` at bottom of pool.rs.

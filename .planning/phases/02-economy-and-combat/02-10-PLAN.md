Two-part change: shader uniform support + Rust-side buffer and pipeline.

**Part A: Shader (objects_tex.wgsl)**

The building shader uses groups 0 (transforms+lighting), 1 (texture), 2 (shadow). Add a group 3 for ghost overlay. Add at the top of the shader after group 2 bindings:

```wgsl
// Group 3: Ghost overlay (optional, for placement preview)
struct GhostParams {
    tint: vec3<f32>,
    alpha: f32,
};
@group(3) @binding(0) var<uniform> ghost: GhostParams;
```

Modify `fs_main` to multiply the final color by ghost.tint and set alpha to ghost.alpha. At the end of fs_main, before each return statement, multiply:
- Change `return vec4<f32>(color.rgb * brightness * shadow_factor * in.viewport_fade, color.a);` to:
  `return vec4<f32>(color.rgb * brightness * shadow_factor * in.viewport_fade * ghost.tint, ghost.alpha);`
- Also update the untextured fallback (tex_id < 0 branch):
  `return vec4<f32>(vec3<f32>(0.6) * brightness * shadow_factor * in.viewport_fade * ghost.tint, ghost.alpha);`

For normal (non-ghost) rendering, ghost.tint = vec3(1.0, 1.0, 1.0) and ghost.alpha = 1.0 (identity).

8. Replace the ghost preview placeholder (lines 2244-2258) with actual rendering:
```rust
if let Some(ref ghost) = frame.ghost_preview {
    let ghost_alpha: f32 = 0.5;
    let ghost_tint: [f32; 3] = if ghost.valid {
        [0.3, 1.0, 0.3]
    } else {
        [1.0, 0.3, 0.3]
    };
    // Write ghost uniforms
    queue.write_buffer(
        ghost_buf,
        0,
        bytemuck::cast_slice(&[ghost_tint[0], ghost_tint[1], ghost_tint[2], ghost_alpha]),
    );
    // Draw ghost building mesh with alpha blending
    // (The pipeline already has ALPHA_BLENDING if we set it, or we use a separate ghost pipeline)
    if let Some(ref model) = self.model_buildings {
        model.draw(&mut render_pass);
    }
    // Restore normal uniforms after ghost draw
    queue.write_buffer(
        ghost_buf,
        0,
        bytemuck::cast_slice(&[1.0f32, 1.0, 1.0, 1.0]),
    );
}
```

IMPORTANT: The normal building draw (line 2232) needs the ghost uniform set to identity [1.0, 1.0, 1.0, 1.0] so non-ghost buildings render normally. Write identity values before the normal draw call.

9. For the ghost draw to use alpha blending, create a second pipeline variant `ghost_building_pipeline` that is identical to `building_pipeline` but with `blend: Some(wgpu::BlendState::ALPHA_BLENDING)` instead of `BlendState::REPLACE`. Use this pipeline for the ghost draw call only.

NOTE: An alternative simpler approach is to use a single pipeline with ALPHA_BLENDING for all buildings (since alpha=1.0 with ALPHA_BLENDING is equivalent to REPLACE for opaque geometry). This avoids creating a second pipeline. Choose whichever approach is cleaner.
